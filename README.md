# Adventure-Works-Sales-Dashboard
# 🚀 Adventure Works Sales Dashboard (Power BI)

## 📊 Overview
This project showcases an interactive Sales Dashboard built using Power BI, based on the Adventure Works dataset. The dashboard provides a comprehensive view of sales performance, customer insights, and product trends to support data-driven decision-making.

## 🎯 Key Features

- 📈 **Executive Summary**
  - Total Revenue: 24.91M
  - Total Orders: 25.16K
  - Total Returns: 1.83K
  - Total Customers: 18K

- 🎯 **Order vs Target Analysis**
  - Visual gauge to track performance against targets

- 🛍️ **Top Products by Orders**
  - Highlights best-selling products

- 👥 **Top Customers by Profit**
  - Identifies high-value customers

- 🧩 **Category-wise Revenue Breakdown**
  - Bikes, Accessories, Clothing

- 📅 **Monthly Sales Trends**
  - Revenue vs Orders comparison over time

- 🔍 **Interactive Filters (Slicers)**
  - Country
  - Date Range
  - Category

## 💡 Insights Derived

- Bikes contribute the majority of revenue
- Seasonal trends observed in mid-year performance dips
- A small group of customers drives significant profit
- Accessories have high order volume but lower revenue
